# 🐬 Dolfyn - Treinamento & Desenvolvimento Corporativo

Uma landing page premium, responsiva e de alta conversão desenvolvida para a **Dolfyn**, focada em impulsionar resultados de negócios através do desenvolvimento humano.

O projeto apresenta um design moderno com estética "Dark Mode" elegante, Glassmorphism e animações fluidas de scroll para uma experiência de usuário imersiva.

## ✨ Funcionalidades e Diferenciais

* **Design de Alto Padrão:** Interface limpa, contrastes estudados e uso estratégico da cor de destaque (Laranja Dolfyn).
* **Animações GSAP:** Animações de entrada, scroll reveal e uma timeline vertical interativa na seção de depoimentos.
* **Carrossel Infinito com CSS Puro:** Galeria de clientes otimizada com performance impecável.
* **Hub de Contato Glassmorphism:** Cards com efeito de vidro jateado e hover states dinâmicos com cores oficiais das marcas.
* **100% Responsivo:** Menu hambúrguer lateral dinâmico para mobile e adaptação perfeita em qualquer resolução de tela.
* **Botão "Voltar ao Topo":** Melhora na usabilidade para navegação em landing pages longas.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica.
* **CSS3:** Flexbox, CSS Grid, Variáveis (Custom Properties), Animações Keyframes e Glassmorphism.
* **JavaScript (Vanilla):** Lógica de interações do menu mobile e botão "Back to Top".
* **GSAP (GreenSock):** Biblioteca de animação avançada para o scroll e elementos da página.
* **FontAwesome (v6):** Ícones vetoriais dinâmicos.
